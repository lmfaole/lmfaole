# Work

## Frontend developer for [Fremtind](https://fremtind.no)

From 2023

Working together with designers on [Jøkul](https://jokul.fremtind.no), Fremtind's open source design system. In addition to work on the components themselves we document related best practices.

The components are designed in Figma, developed with React and SCSS, versioned with Changesets, published to NPM Registry and Github Packages. Documented with Sanity as the CMS, and Next.js as the server side front-end framework.

Key technologies: React, Sanity, SCSS, Remix, Next.js, Figma.

Key insights: frontend development, design systems, documentation, accessibility.

[Full commit log on the Jøkul-repository](https://github.com/fremtind/jokul/commits?author=lmfaole).

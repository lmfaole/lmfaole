## Når bør du bruke en ikonknapp?

Ikonknapper passer når handlingen er velkjent (f.eks. lukk, søk, rediger) og det
ikke er behov for synlig tekst.

Bruk vanlig `Button` med `variant="ghost"` og `icon`, og gi alltid en tydelig
`aria-label`.

## Anbefalt

```tsx
import { Button } from "@fremtind/jokul/button";
import { Icon } from "@fremtind/jokul/icon";

<Button
    variant="ghost"
    aria-label="Søk"
    icon={<Icon>search</Icon>}
    onClick={handleSearch}
/>;
```

## Unngå

- Ukjent ikon uten forklaring
- Manglende `aria-label`
- Primær-variant for lavprioritert handling (bruk heller `ghost`)
